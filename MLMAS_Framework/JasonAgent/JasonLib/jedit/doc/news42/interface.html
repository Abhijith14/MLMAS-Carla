User Interface

jEdit now saves the configuration of open views between sessions; this includes including their position, size and any splits. This information is stored in the perspective.xml file in the settings directory.

Dockable windows can now be moved between panes without having to open the Utilities>Global Options dialog box. Just right-click on the dockable window's button, or for floating windows, click the small triangle in the top left corner of the window. In both cases this displays a menu where you can pick a new docking location for the window.

A new Utilities>Repeat Last Action command (shortcut: Control-Space) repeats the most recently invoked command.

The status bar can now optionally display a simple clock. This can be enabled in the Status Bar pane of the Utilities>Global Options dialog box
